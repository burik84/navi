    Нави - это проект, где собрана информация по оборудованию КИП, АСУТП, схемам, инструкциям и т.д.

    Важно!

    Внутри папки navi, в связи с тем что используются относительные ссылки на название файлов, запрещается

    • переносить папки и файлы;
    • переименовывать папки и файлы.

    Как установить на ...

    Windows
    1. Скопировать папку navi на свой ПК в любое удобное место.
    2. В папке navi открыть файл index.html.
    3. Пользоваться.
    Android

    Подготовка к использованию. Для работы на мобильном устройстве необходимо установка сервера и браузера, например Tiny Web Server Free и браузер Chrome или использовать другие (в помощь и ).

    Для уменьшения места перед копированием на мобильное устройство рекомендую заранее сделать копию navi и в ней удалить ненужную информацию из папки source - файлы и папки, при этом помнит о важном).

    1. Скачать подготовленную папку navi на карту памяти в любое место мобильного устройства - планшет или телефон .
    2. Открыть приложение Tiny Web Server Free, настроить и запустить.
    3. Открыть браузер, ввести в строку адреса наш ipaddress:port из настроек приложения Tiny Web Server Free.
    4. Пользоваться.

    Ключевые правила безопасности:

    Сокрытие информации

    Сокрытие информации об авариях, пожарах, инцидентах, фактах производственного травматизма, потенциально опасных происшествиях

    Наряд-допуск

    Проведение работ повышенной опасности без наряда-допуска, а также нарушение его требований

    Блокировки без разрешения

    Отключение или нарушение целостности блокировок, противоаварийной автоматической защиты и устройств обеспечения безопасности на действующем оборудовании без соответствующего письменного разрешения

    Алкоголь

    Появление на территории предприятия в состоянии алкогольного, наркотического или иного токсического опьянения

    Курение и огонь

    Курение на территории предприятия вне специально отведенных для этой цели мест или использование открытого огня без специального разрешения
