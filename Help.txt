ВАЖНО!
Запрещено внутри папки navi любое перемещение папок или переименование папок и файлов. Связано с относительными путями к файлам для работы навигатора. 

Инструкция пользования на ПК Widows.
Скачать папку с navi на диск. 
Открыть файл index.html. 
Пользоваться.

Инструкция пользования на мобильных устройствах Android.
Для работы необходимо установка сервера. Тестирование проводилось на приложение Tiny Web Server Free и браузер Chrome.
Скачать navi на мобильное устройство - планшет или телефон на карту памяти в любое место. Для уменьшения место можно удалить папки/файлы из папки "Source" - информация разбита тематически. После удаления соответственно в НАВИГАТОРЕ не будет доступна удаленная информация.
Скачать приложение Tiny Web Server Free  с Google Play(можно другое, но со всеми проблемами решаете самостоятельно, в помощь Яndex.Алиса и Google).
Установить приложение. Настроить. Запустить.
Открыть браузер. Ввести в строке адреса ipaddress:port.
Пользоваться.
Примечание: после перезапуска сервера, может меняться ip address.
